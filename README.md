
# HabitPet 🐾

HabitPet is a gamified productivity and habit tracking web app built using React, Vite, Tailwind CSS, and Framer Motion.

Users can:

* Track daily habits
* Earn XP
* Level up pets
* Use a Pomodoro focus timer
* Unlock rewards
* Build streaks
* View analytics charts
* Switch between dark/light themes

The project focuses on making productivity fun through gamification.

---

# 🚀 Features

## ✅ Habit Tracking

* Add daily habits
* Mark habits as completed
* Delete habits
* LocalStorage persistence

## 🎮 Gamification System

* Gain XP for completing tasks
* Level progression system
* Animated pet evolution
* Unlockable rewards

## 🍅 Pomodoro Focus Timer

* 25-minute focus sessions
* Session tracking
* XP rewards for completed sessions

## 🔥 Streak System

* Build productivity streaks
* Streak counter tracking

## 📊 Analytics Dashboard

* Interactive charts using Recharts
* XP tracking
* Session tracking
* Streak visualization

## 🌙 Dark / Light Mode

* Toggle between themes
* Smooth UI transitions

## 🎉 Achievement Effects

* Confetti celebrations
* Achievement popup animations

## ✨ UI & Animations

* Responsive design
* Framer Motion animations
* Hover effects
* Smooth transitions

# 📦 Installation

Clone the repository:

```bash
git clone <your-repo-link>
```

Go into the project folder:

```bash
cd habitpet
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

# ⚙️ Required Packages

Install these packages if needed:

```bash
npm install framer-motion recharts react-icons react-confetti
```

Build command:

```bash
npm run build
```
